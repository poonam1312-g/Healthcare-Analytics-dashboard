**Healthcare Analytics Dashboard
Project Description**

The Healthcare Analytics Dashboard is a data visualization project designed to analyze healthcare data and provide insights into patient trends, hospital departments, treatment outcomes, and revenue.

The dashboard helps healthcare administrators and analysts understand patient distribution across diseases, departments, and cities. It also tracks recovery rates, treatment costs, and admission trends over time.

By using interactive visualizations, users can explore the data, identify patterns, and support better decision-making in healthcare management.

**Tech Stack**

The following tools and technologies were used to build this project:

Power BI – Used for creating interactive dashboards and visualizations

Microsoft Excel – Used for data storage, cleaning, and preprocessing

Power Query – Used for transforming and preparing the dataset

GitHub – Used for project version control and portfolio presentation

**Data Source**

The dataset used in this project is a healthcare patient dataset containing information about patient demographics, diseases, hospital departments, treatment outcomes, and costs.

Dataset includes the following fields:

Patient_ID – Unique identifier for each patient

Gender – Male or Female patient information

Disease – Type of disease diagnosed (Heart Disease, Asthma, Cancer, etc.)

Department – Hospital department responsible for treatment

City – Location of the hospital or patient center

Treatment_Cost – Cost of the medical treatment

Insurance Status – Whether the patient has insurance coverage

Outcome – Patient status such as Recovered, Critical, or Under Treatment

Year – Admission year for trend analysis

This dataset helps analyze healthcare patterns and operational performance.

Disease Analysis

A bar chart shows the number of patients by disease type, helping identify the most common medical conditions such as:

Heart Disease

Flu

Stroke

Fracture

This helps hospitals understand which diseases require more resources.

**Gender Distribution**

A donut chart displays the distribution of male and female patients, helping analyze gender-based healthcare trends.

Department Analysis

A department-wise chart shows the number of patients treated in each department, including:

General Medicine

Orthopedics

Cardiology

Oncology

Pediatrics

Neurology

This helps identify departments with the highest workload.

**Patient Admission Trend**

A line chart displays patient admission trends by year, allowing hospitals to track changes in patient volume over time.

Treatment Cost Analysis

A pie chart compares total treatment cost based on insurance status, helping understand:

Cost distribution for insured patients

Cost distribution for uninsured patients

This provides insights into healthcare financial patterns.
